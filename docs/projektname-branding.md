# 🏷️ Projektname & Branding – MaintCloud AI

---

## 🧠 Projektname

### MaintCloud AI

Der Name setzt sich aus drei zentralen Komponenten zusammen:

- **Maint** → Maintenance (Wartung)
- **Cloud** → Cloud-basierte Architektur
- **AI** → Intelligente Analyse / zukünftige KI-Funktionen

👉 Bedeutung:  
Ein intelligentes, cloudbasiertes Wartungssystem für industrielle Anwendungen.
